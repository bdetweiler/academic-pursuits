Polynomials

You need to create a polynomial data type. You need to be able to add, subtract, and multiply your polynomials. In addition you will need to be able to output them, and add terms to them.

Keep in mind that polynomials are a collection of terms, which include a coefficient, variable, and exponent. The number of terms must be dynamic. To make the problem easier, the variable will always be 'x'.
