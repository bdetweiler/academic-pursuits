Graphs

This program needs to compute the shortest path for a graph. I would recommend using Dijkstra's for this problem.

This program needs to read the input from a file. The file is format is a path to compute and the set of edges. Each edges is in the form of source node, destination node, and distance. For this problem the graphs are not directed, and node names may be contigious non-whitespace symbols.

The output needs to give the distance and the path from source the destination.
